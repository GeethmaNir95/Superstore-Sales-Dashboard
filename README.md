# Superstore Sales Dashboard
## Overview
This project focuses on delivering the key insights and measures of the  sales performance of Superstore during the period from 2014-2017.The project aims to find out the key performance indicators (KPIs) such as Total Sales, Profit Percentage (%) across all time and the Total Quantities as well as analysis of profit and  sales across different geographical locations, segements and product categories and sub categories.
